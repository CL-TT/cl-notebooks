# 像素的概念

## 1.CSS像素

1. css像素（设备的逻辑像素， 或者说是独立像素）
2. 他的大小是不固定的， 他的大小是取决于**设备像素**的



## 2.设备像素

1. **设备像素又称物理像素**， 也就是真实存在的，设备出厂之后就不会改变了
2. **设备的分辨率就是设备像素**，是由一个点一个点组成的（点的单位是pt）
3. 比如分辨率为750 * 1334   =>      750pt  *  1334pt



## 3.屏幕尺寸

1. 屏幕尺寸指的是屏幕对角线的长度

2. 1英寸（inch） =  2.54厘米（cm） 
3. **屏幕尺寸 = 屏幕斜边的像素 / PPI**



## 4.像素密度（PPI）

1. **像素密度，指的是每英寸面积上的设备像素的数量**，这个是出厂之后就定好的
2. **PPI = 屏幕斜边的像素 / 屏幕尺寸**
3. 在同一尺寸下，PPI提高一倍， 那么像素会提高四倍
4. PPI的值会决定图片视频的清晰度



## 5.像素比（DPR）

1. **DPR = 设备像素 / CSS像素**
2. 本质： 一个CSS像素会占据几个设备像素
