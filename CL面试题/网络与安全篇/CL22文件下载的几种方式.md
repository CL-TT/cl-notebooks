# 文件下载的几种方式

## 1. 如果不传Token的话

```js
那么直接使用a元素就可以了

<a download="a.pdf" href="地址" />
    
    
那么它的过程浏览器发送请求，服务器把数据返回给浏览器，浏览器这个时候是不做任何处理的，直接把数据流向文件的
```

## 2. 如果要传Token的话

```js
这种方式其实是不好的，如果下载的文件过大，那么就会导致用户一直在等待，造成不好的体验


const res = await fecth('', {
    headers: {
        token: ''
    }
})

const blob = await res.blob()

const url = URL.createObjectURL(blob)

const a = document.createElement('a')

a.herf = url

a.download = 'a.pdf'

a.click()
```
