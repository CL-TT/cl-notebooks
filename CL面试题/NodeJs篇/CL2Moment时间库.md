# Moment时间库详解

## 1.理解UTC(世界协调时)时间和北京时间

1. 北京时间要比UTC时间快8个小时， 但是这两个时间是同一刻时间，只是表示形式不一样
2. 比如说UTC时间现在为1970-01-01 00:00:00，那么北京时间为1970-01-01 08:00:00



## 2.时间戳

#### 2.1. 什么是时间戳

时间戳是某个UTC时间到UTC1970-01-01凌晨经过的毫秒数

使用时间戳可以消除这两种时间上的差异

北京时间 1970-01-01 08:00:00    =>   时间戳 0

UTC时间 1970-01-01 00:00:00    =>   时间戳  0



## 3.对于服务器和客户端的影响

1. 服务器可能会部署到世界的任何位置
2. 服务器内部应该统一使用UTC时间或者是时间戳，包括数据库
3. 客户端应该把UTC时间和时间戳转化为本地时间进行展示



## 4.本地时间和utc时间的相互转换

#### 1.utc时间转为本地时间

```js
const time = moment.utc();         //这是得到一个utc时间
const localTime = time.local().format('YYYY-MM-DD HH:mm:ss');
```



#### 2.本地时间转为utc时间

```js
const time = moment();     //这是得到一个本地的时间
const utcTime = time.utc().format('YYYY-MM-DD HH:mm:ss');
```
