# 如何画一条0.5px的线

### 1.采用transform中的scale， 缩放属性

```javascript
transform: scale(0.5, 0.5);
```



### 2.采用meta viewport

```javascript
<meta name="viewport" content="width=device-width, initial-scale=0.5, minimum-scale=0.5, maximum-scale=0.5"/>
    
    
    1. initial-scale: 初始的缩放比例
    2. mininum-scale: 最小的缩放比例
    3. maxinum-scale: 最大的缩放比例
    
这样就可以达到效果， 1px 变成 0.5px,    viewport 只针对移动端 所以这样只适用于移动端
```
