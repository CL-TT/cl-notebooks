# 行级元素和块级元素

## 1. 行级元素

1. 行级元素是不可以设置设置宽高的，不能独占一行。

2. 常见的行级元素

   1.span

   2.a

   3.strong

   4.b

   5.em

   

   ## 2.块级元素

   1. 块级元素独占一行，可以为其设置宽高属性

   2. 常见的块级元素：

      1.div

      2.ul

      3.li

      4.p

      5.h1-h6

      6.dl

      7.dt

      8.dd

   ## 3.行级块元素

   1. 既有行级元素的特点又有块级元素的特点(可以设置宽高， 但又不会独占一行)

   2. img ,      input
   
   3. 凡是带有inline特性的元素， 都带有文字特性，就会留有空隙  一般为8px  
   
       font-size: 0来解决
