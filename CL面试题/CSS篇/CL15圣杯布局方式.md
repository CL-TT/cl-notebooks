# 圣杯布局方式

## 1.什么是圣杯布局

1. 上部也就是header和下部也就是footer各自占有屏幕的所有宽度
2. 上下部之间的部分是一个三栏布局
3. 三栏布局两侧的宽度不变，中间部分自动填充整个区域
4. 中间部分的高度是三栏中最高的区域的高度





## 2.两种解决方式

```html
让main的内容最先展示

1. float的方式

<div id="wrap">
    <div class="header">header</div>
    <div class="main">
        <div class="main-div item">main</div>
        <div class="left-div item">left</div>
        <div class="right-div item">right</div>
    </div>
    <div class="footer">footer</div>
</div>


<style>
    div{
        margin: 0;
        padding: 0;
    }
    
    .header, .footer{
        width: 100%;
        height: 60px;
    }
    
    .header{
        background: red;
    }
    
    .footer{
        background: green;
    }
    
    .main{
        padding: 0 300px 0 200px;
    }
    
    .main::after{
        content: '';
        display: block;
        clear: both;
    }
    
    .item{
        float: left;
        height: 600px;
        position: relative;
    }
    
    .main-div{
        width: 100%;
        background: pink;
    }
    
    .left-div{
        width: 200px;
        background: red;
        margin-left: -100%;
        left: -200px;
    }
    
    .right-div{
        width: 300px;
        background: yellow;
        margin-right: -300px;
    }
</style>



2. display: flex弹性布局的方式

<style>
    .header, .footer{
        width: 100%;
        height: 60px;
    }
    
    .header{
        background: red;
    }
    
    .footer{
        background: green;
    }
    
    .main{
        display: flex;
    }
    
    .left-div{
        width； 200px
        order: -1;
    }
    
    .right-div{
        width: 300px;
    }
    
    .main-div{
        width: 100%;
    }
    
    .item{
        height: 600px
    }
</style>
```
