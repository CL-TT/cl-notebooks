# 清除浮动的几种方式

## 1.清除浮动主要解决什么问题

父元素因为子级元素浮动引起的内部高度为0的问题



## 2.清除浮动的几种方法

```css
1. 额外标签法(在最后一个浮动标签后面，新加一个标签，给其设置clear：both)

缺点: 添加无意义的标签，语义化较差

<div class="clear"></div>

.clear{
    clear:  both
}


2. 让父元素触发BFC

可以给父元素设置  
overflow: hidden   
display: inline-block
float: left/right
position: absolute

缺点: 添加这些属性时，就要为这些属性带来的副作用买单


3. 利用after伪元素清除浮动， 在父元素上加上伪元素(推荐使用这种方法)

这种方式需要注意，必须要给伪元素设置成块级元素
.wrap{
    content: '',
    display: block,
    clear: both
}

```
