# 元素隐藏三种方式的区别

## 1.display: none

```javascript
//display: none
//隐藏元素,元素隐藏后不会占位置，元素的宽高和其他属性都会失效
```



## 2.visibility: hidden

```javascript
//visibility: hidden
//同样是隐藏元素，但元素隐藏后会占据着原来的位置，元素的宽高和其他属性都不会失效
```



## 3.opacity: 0

```javascript
//opacity：0 
//元素的透明度为0，同样起到隐藏元素的作用，但元素隐藏后，会占着原来的位置，宽高都不会失效
```
