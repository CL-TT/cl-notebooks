# 数组中的方法

## 1.可以改变原数组的方法

### 1.1.往数组里面添加数据

1. Array.push()
2. Array.unshift()
3. Array.splice()

```javascript
//1. *****************array.push(num)   往数组的末尾添加数据******************
let array = [1, 2, 3];

//num = 6, 并且数组被改变了 [1, 2, 3, 6]
let num = array.push(6);

//2. *******************array.unshift(num)   往数组的最前端添加数据******************
let array = [1, 2, 3];

//num = 4, 并且数组被改变了  [4, 1, 2, 3]
let num = array.unshift(4);
//当插入多个数据时，会看作一个整体  [5, 6, 7, 1, 2, 3]
let num1 = array.unshift(5, 6, 7)

//3. *************array.splice(开始索引(可以用负索引)， 要删除数据的数量， 要添加的数据)***************
let array = [1, 2, 3, 4, 5];

//添加数据的话num=[], 原数组被改变了[1, 8, 2, 3, 4, 5]
let num = array.splice(1, 0, 8);
```



### 1.2.删除数组里面的数据

1. Array.pop()
2. Array.shift()
3. Array.splice()

```javascript
//1. ***************************array.pop()删除数组的最末尾数据***************************
let arr = [1, 2, 4];

//num = 4  原数组会被改变[1, 2]
let num = arr.pop();

//2. ******************array.shift()删除数组的最前端的数据*******************
let arr = [1, 2, 3, 4];

//num = 1 原数组会被改变[2, 3, 4]
let num = arr.shift();

//3. *************array.splice(开始索引， 删除数据的个数)******************
let arr = [1, 2, 3, 4, 5];

//返回值num = [2, 3]被删掉的数据    原数组会被改变[1, 4, 5]
let num = arr.splice(1, 2);
```



### 1.3.其他方法

1.  Array.fill(数据) 数组填充：将数组的所有项，填充为指定的数据

2.  Array.fill(数组, 开始下标): 将数组从开始下标起，到数组的末尾，填充为指定的数据

3.  Array.fill(数组, 开始下标，结束下标): 将数组从开始下标起，到数组的结束下标（取不到），填充为指定的数据



## 2.不会改变原数组的方法

1. Array.slice()
2. Array.indexOf()
3. Array.lastIndexOf()

```javascript
//1.数组的截取  Array.slice(起始位置， 终止位置)  取不到终止位置，  把取出来的数据放入一个数组中并返回
let arr = [1, 2, 3, 4, 5];

//num = [2, 3, 4]
let num = arr.slice(1, 4);
let num = arr.slice(1) //传一个参数，直接从开始位置到结束  [2, 3, 4, 5]
//索引可以取负数， 就是从后面开始数，   只能从理论上从左往右数
slice(-1, -3)//这种情况不行      
slice(-3, -1)//这种情况可以

//2.清空数组
Array.splice(0, 数组.length);

Array.length = 0;


//3.查找数组中某一项的下标
//3.1     Array.indexOf('')    返回的是这个数据的索引值
let arr = [1, 2, 3, 4];
arr.indexOf(2);     //会返回索引1，   找不到就返回-1

//3.2     Array.lastIndexOf('')  查找的是最后一个匹配的下标
let arr = [1, 2, 3, 4, 2];
arr.lastIndexOf(2);   //会返回索引4



```
