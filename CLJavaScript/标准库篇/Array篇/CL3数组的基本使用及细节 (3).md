# 数组的基本使用以及细节

## 1.数组的基本使用

1. 数组是用来保存大量数据的， 数组是对象的一种，具有对象的特点

   ### 1.1.创建数组的方式

   ```javascript
   //第一种
   let array = new Array(10);   //这是创建数组长度为10的数组，还没有赋值
   
   //这是创建一个赋值的数组，长度为4，索引和值对应0-1，1-2，2-3，3-4
   let array2 = new Array(1, 2, 3, 4);  
   
   //第二种
   let array3 = [1, 2, 3];   //直接创建一个数组
   
   
   ```



###      1.2.数组的本质

1. 数组的本质是一个对象

   2. length属性：数组的长度，会自动变化，值为最大下标+1

      ```javascript
      //1.第一种情景
      var arr = [1, 2, 3, 4, 5];
      
      arr.abc = 'caolei';  //这样写length也不会改变    length = 5
      
      console.log(arr.abc) //可以打印出 'caolei'
      console.log(arr[5]) //是打印不出来 'caolei'
      
      //2.第二种情景
      var arr = [1, 2, 3, 4,];
      
      arr[50] = 50;
      
      //这时候length是等于 最大下标 + 1     length = 50 + 1 = 51
      
      
      //3.第三种情景
      let arr = [1, 2, 3];
      arr[1] = 10;
      arr["1"] = 20;
      console.log(arr[1], arr['1']);   //两个都会打印20
      
      //因为arr[1]和arr['1']是相同的效果
      ```

      

3. 数字字符串属性：叫做下标，也叫做索引，相当于数组中每个数据的编号，下标从0开始排列


​      连续下标的取值范围：0 ~ length -1，如果给length直接赋值，会导致数组可能被截断

​      实际开发中，不要给length赋值。
