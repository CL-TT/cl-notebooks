# 作用域详解

## 1.作用域的概念

作用域表示一个代码区域，也表示一个运行环境



## 2.js中的两种作用域(es6除外)

### 2.1.全局作用域

在全局作用域中声明的变量，会被提升到脚本块的顶部，并且会成为全局对象的属性。

```javascript
var a = 12;
console.log(a);

//全局都可以访问到的
```

### 2.2.函数作用域

在函数作用域中声明的变量，会被提升到函数的顶部，并且不会成为全局对象的属性.

**因此，函数中声明的变量不会导致全局对象的污染**

**尽量的把功能封装在函数中**

```javascript
var age = 20;
function a(){
    var name = 'caolei';
    console.log(name);
    console.log(age);   
}
a();  //函数里面是可以访问到全局变量的
console.log(name);    //会报错，全局是访问不到函数作用域里面的变量的
```



## 3. 函数变成表达式后的结果

1.但是，当函数成为一个表达式时，它既不会提升，也不会污染全局对象。

2.将函数变为一个函数表达式的方式之一，将函数用小括号括起来。然而，这样一来，函数无法通过名称调用。

3.如果书写一个函数表达式，然后将立即调用，该函数称之为立即执行函数 IIFE（Imdiately Invoked Function Expression）。

4.由于大部分情况下，函数表达式的函数名没有实际意义，因此，可以省略函数名。没有名字的函数，称之为匿名函数。

```javascript
(function a(){})   //这样就变成了一个函数表达式

(function a(){})()   //立即执行函数

(function (){})()   //匿名函数
```
