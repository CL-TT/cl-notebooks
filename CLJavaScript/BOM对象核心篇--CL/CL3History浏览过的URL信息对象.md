# History浏览过的URL信息对象

## 1. History的属性

#####   1. history.length    返回浏览器历史列表中的 URL 数量







## 2. History的方法

#####   1. history.back()    浏览器后退

```javascript
//后退到上一个页面
history.back();    
```



#####  2. history.forward()     浏览器前进，加载列表中的下一个URL

```javascript
history.forward();
```



#####  3. history.go()   加载到history列表中的某个具体的页面

``` javascript
//数字可以为负数
history.go(数字); 
```
