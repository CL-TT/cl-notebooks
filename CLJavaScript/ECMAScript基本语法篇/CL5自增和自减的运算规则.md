# 自增和自减的运算规则

## 1. 优先级原则

++ --    >     *   /    %    >     +   -   

## 2. 自加和自减

```javascript
//1. x++    先用x再加
//2. ++x    先加再用
```

## 3. 运算规则

1. 从左到右依次查看
2. 如果遇到操作数将数据的值直接取出
3. 如果遇到相邻的两个运算符，左边的运算符优先级大于等于右边，则直接可以计算。

```javascript
var x = 1;
var y = x + x++ * (x = x + x++ * ++x) + x

//1. 从左到右依次查看  x = 1
//2. y = 1 + 1 * (x = x + x++ * ++x) + x    x = 2
//3. y = 1 + 1 * (x = 2 + 2 * 4) + x    x = 4
//4. y = 1 + 1 * 10 + x            x = 10
//5. y = 1 + 10 + 10 = 21
```
