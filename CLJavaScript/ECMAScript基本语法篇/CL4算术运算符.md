# 算术运算符

## 1. 表达式

```javascript
1. //表达式 = 操作符 + 操作数
2. //操作符: 指代的就是运算符
3. //操作数：指代的就是数据

4. //每一个表达式都有一个运算结果， 该结果叫做返回值， 返回值的类型叫做返回类型。

var a = 3;   //返回结果为undefined     打印a还是3

a = 4  //返回结果是4

console.log();   //返回结果为undefined

function a(){};    //返回结果为undefined

a();    //如果函数里面有返回值返回结果就是那个返回值，   如果没有返回值返回结果就是undefined
```

### 2.3 其他类型进行算术运算

##### 2.3.1 除加号之外的算术运算

```javascript
//1. 原始类型
// boolean    true =>  1    false => 0
// string{
//  1.字符串中是一个正确的数字  '123' =>  123
//  2.不是一个正确的数字  '12as' => NaN
//  3. 'Infinity' => Infinity     不能把字符串内部的东西当做表达式
//}
// null => 0
// undefined => NaN

//2. 对象类型
//对象类型先转为字符串类型 => [object Object]， 再把字符串类型转为数字类型
//对象类型 => NaN

//3. NaN和任何数字进行运算都得 NaN

//4. +'' => 0    如果有空格会忽略字符串文字前后的空格，中间的空格不会忽略
//   +' 1   1   ' => NaN    中间有空格所以不是正确的数字
```

##### 2.3.2 带加号的算术运算

```javascript
//1. 加号一边带有字符串，  都是字符串的拼接
'sd' + 2 = 'sd2'
'12' + 2 = '122'
'12' + '2' = '122'
'true' + 2 = 'true2'
null + '3' = 'null3'

//2. 没有字符串但是有对象
({name: 'caolei'} + 123) =>  '[object Object]' + 123  =>  '[object Object]123'
```
