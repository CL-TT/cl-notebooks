# 提测与部署

1.它依托与 winscp 这个软件

## 1. 部署部分

```js
1. 在home文件夹下面的front文件夹
2. 在front文件夹下面找到对应的项目文件夹
3. 把里面的文件全部删除掉
4. 在自己电脑的对应文件夹下把dist文件夹里面的文件copy到对面
```



## 2. 提测部分

```js
1. 在home文件夹下面的bosssoft文件夹
2. 在bosssoft文件夹下面找到对应的项目
3. 在对应的项目下面创建，对应的日期文件夹
4. 在对应的日期文件夹下面创建对应的项目文件夹
5. 然后把dist文件夹的内容放入其中
```



## 3. 部署代理项目的步骤

```js
1. 在 etc/nginx/conf.d/default.conf 配置代理

2. 重启nginx

3. 在对应的home/front/   文件夹里面创建对应的项目
```
