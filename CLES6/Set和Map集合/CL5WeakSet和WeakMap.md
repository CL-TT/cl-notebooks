# WeakSet和WeakMap

## 1.演示一下垃圾回收机制

```js
let obj = {
    name: 'caolei'
}

//这时候在内存中会开辟一个空间obj这个变量保存对象的地址

//obj对象地址  =>   obj

const s = new Set();
s.add(obj);  //经过此操作，s集合中也保存了这个对象的地址，指向这个对象

obj = null;  //当obj置为null的时候

//因为set集合中还有地址指向这个对象，所以垃圾回收机制并不会回收这个对象，在set集合中还是能够访问到这个对象的

```



## 2.WeakSet

```js
1. WeakSet的功能几乎和set功能类似，不同的是

不同点： 
1. 它内部存储的对象地址不会影响垃圾回收机制
2. 只能添加对象
3. 不能遍历(不是可迭代对象)，没有size属性，没有forEach方法

4. 它可以观察哪些对象被垃圾回收了
```



## 3.WeakMap

```js
1. 类似于map集合

不同点：
1. 他的键只能是对象
2. 他的键存储的地址不会影响垃圾回收
3. 不能遍历(不是可迭代的对象)，没有size属性，没有forEach方法
```
