# ES概述

## 1.区分开ES， JAVSCRIPT,    NODEJS

1. ES => ECMAScript,    他是一个语言标准，  就是说话的方式
2. JavaScript：就是 es + webapi   他是运行在浏览器端的语言， 他使用ECMAScript标准
3. NodeJs：就是es + nodeapi   他是运行在服务器端的语言， 他同样使用ECMAScript标准

## 2.ES6的重要版本

1. ES3.0     1999年
2. ES5.0      2009年
3. ES6.0      2015年     =》  ES2015
4. 在3.0  =>  5.0的过渡期，经历了十年, 可以说在那段时期是js的黑暗时期，十年的时间停滞不前，所以当时是积累了很多的问题的

## 3.ES6为什么这么重要

ES6.0解决了JS无法开发大型应用的语言层面的问题， 让JS成为了正规语言
