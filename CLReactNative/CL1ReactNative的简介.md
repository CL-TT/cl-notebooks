# ReactNative的简介

## 1. 什么是ReactNative

ReactNative是FaceBook于2015年开源的跨平台移动应用开发框架，目前支持iOS和安卓两大平台





## 2. 特点

1. 使用react创建
2. 安卓和iOS原生应用
3. 预览速度快
4. 无缝跨平台



## 3. 搭建开发环境

1. 学习阶段，快速体验RN的开发，可以直接使用简易的沙盒环境
2. 如果要做完整的上限应用开发，可以搭建完整的原生环境

```js
沙盒环境为例

1. 安装expo-cli脚手架工具 npm i expo-cli -g

2. 安装之后可以查看脚手架版本  expo -V

3. 创建项目expo init 项目名称
```

3. 在真机上运行的时候，需要把网络切换到专用网络，   还要确保真机和电脑在同一网络上
