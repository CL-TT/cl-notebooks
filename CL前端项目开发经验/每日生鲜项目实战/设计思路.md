# 每日生鲜项目实战

## 1.创建项目并配置

1. 创建项目利用vue脚手架创建

2. **配置lib-flexible解决移动端适配的问题**

   #### lib-flexible

   1.给元素添加data-dpr属性，并且可以动态改写data-dpr的值

   2.给元素添加font-size属性，并且动态改写font-size的值

   3.安装   npm install lib-flexible -S               生产环境

   4.在main.js中引入这个文件   import 'lib-flexible'

   5.如果项目的index.html有已经有了meta标签，把它删除掉，要不然就起不到作用

   6.lib-flexible会自动在html的head中添加一个`meta name="viewport"`的标签，同时会自动设置html的font-size为屏幕宽度除以10，也就是1rem等于html根节点的font-size 

   #### 2.配置一个插件postcss-px2rem

   1. 在vue.config.js里面进行配置

   ```vue.config.js
   const px2rem = require('postcss-px2rem');
   
   module.exports = {
      css: {
        loaderOptions: {
          postcss: {
            plugins: [
              //如果你想按照设计稿的比例1:1的话，  如果设计稿的宽度为750的话，这里可以写成75
              px2rem({ remUnit: 37.5 })
            ]
          }
        }
      }
   }
   ```

   #### 3.引入vant组件库

   ```
   import Vue from 'vue';
   import Vant from 'vant';
   import 'vant/lib/index.css';
   
   Vue.use(Vant);
   ```
